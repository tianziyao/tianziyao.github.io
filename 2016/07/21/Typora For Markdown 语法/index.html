Typora For Markdown 语法

发表于 2016-07-21   |   分类于 Other   |  

Typora 是一个功能强大的 Markdown 编辑器，Typora 可以插入数学表达式，插入表情，表格，支持标准的 Markdown 语法，可以使用标注，功能非常强悍。

数学表达式

要启用这个功能，首先要到 Preference - Editor 中启用。然后使用$符号包裹Tex命令，例如：$lim_{x \ to \infty} \ exp(-x)=0$。

$lim_{x \ to \infty} \ exp(-x)=0$

下标

下标使用~包裹，例如：H~2~o将产生水的分子式。

H~2~o

上标

上标使用^包裹，例如：y^2^=4将产生表达式。

y^2^=4

插入表情

使用:happy:输入高兴的表情，使用:sad:输入悲伤的表情，使用cry输入哭的表情。

:happy: :cry: :sad:

下划线

使用 HTML 的语法 <u>Underline</u>将产生下划线。

来吧下划线

删除线

使用~~包裹的文本将会具有删除的样式。

来吧删除线

代码

  • 使用` 包裹的内容将会以代码样式显式，例如：

print("Hello World")

  • 输入~~~或者

>

强调

使用两个*或者两个_包裹的内容将会被强调。例如：

我需要被强调

斜体

使用一个*或者一个_包裹的内容将会是斜体显示，如果需要显示*号，使用转义符* 。例如：

我需要斜体和一个*

你可以使用一个标准 URL ，也可以这样给链接起一个名字[连接名]（链接） 例如：

www.baidu.com 百度

目录列表

输入[toc]然后回车，你将得到一个列表，这个列表包含文章内的所有标题。并会自动更新。

[TOC]

水平分割线

使用***或者---，然后回车，来产生水平分割线。
